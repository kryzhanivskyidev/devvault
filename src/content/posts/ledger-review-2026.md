## Security Architecture

Ledger's security is built around a **Secure Element (SE) chip** — the same type used in passports and credit cards. Your private keys never leave this chip, and every transaction must be physically confirmed on the device.

Key security features:
- **CC EAL5+ certified** Secure Element
- **Custom OS (BOLOS)** — isolates each app from others
- **PIN protection** with auto-wipe after 3 wrong attempts
- **24-word recovery phrase** — full BIP39 standard
- **Open-source apps** — wallet apps are publicly auditable

## The Recover Controversy — Where Things Stand

In 2023, Ledger introduced **Ledger Recover** — an optional paid service ($9.99/month) that splits and backs up your seed phrase to three custodians. The crypto community reacted strongly, arguing it proved seed phrases could leave the device.

**In 2026:** Ledger Recover is still optional, the feature is open-source now, and Ledger has clarified the technical implementation. If you don't enable it, your seed phrase never leaves the device.

For most users who don't enable Recover — the security model is unchanged.
